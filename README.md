2. Importar el contenedor Ubuntu: ubuntu.gns3a 
link: https://www.gns3.com/external/download?url=https%3A%2F%2Fraw.githubusercontent.com%2FGNS3%2Fgns3-registry%2Fmaster%2Fappliances%2Fubuntu.gns3a
